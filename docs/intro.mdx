## 如何使用

- 如果你是第一次进入题库，建议先看本页，再按分类进入对应主题页。
- 通过左侧分类切换不同知识域。
- 每个问题块都包含：考察重点、标准答案、代码示例。
- 每个分类页顶部都支持 `全部 / 初级 / 中级 / 高级` 难度筛选。
- 带有运行区域的题目会在文档内部渲染示例，适合边看边验证。

## 内容组织原则

- **分类清晰**：每个主题按常见面试侧重点拆分。
- **答案可落地**：避免只给结论，会补充代码与边界条件。
- **方便扩展**：后续新增分类时，只需补充题库数据并新增一个分类文档即可挂到侧边栏。

## 分类依据

当前分类结构参考了 `MDN`、`web.dev`、`TypeScript`、`React`、`Vue`、`React Native`、`Node.js` 与 `Git` 官方文档的知识分层，覆盖 HTML 与浏览器、样式系统、语言基础、主流框架、工程化、性能、可访问性、测试、中间层与版本协作等高频方向。
