# Vue 专题总览

这个专区聚焦 Vue 的实战知识，适合在准备 Vue 面试、补齐 Composition API 认知，或快速复盘响应式系统时使用。页面会把 Vue 面试里常见的知识点、API 用法和高频踩坑拆成三个子专题，便于搜索引擎和读者快速定位到具体内容。

- 常用知识点：响应式、组件通信、路由与状态管理。
- API 用法：Composition API 与生命周期的高频写法。
- 注意事项：性能、可维护性与常见坑位规避。

## 阅读建议

1. 先看「常用知识点」，建立整体认知。
2. 再看「API 用法」，对照日常开发代码。
3. 最后看「注意事项」，用于代码审查和上线前自检。
