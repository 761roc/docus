# Vue API 用法速查

这页适合在 Vue 面试前快速查阅常用 API 写法，尤其是 `ref`、`reactive`、`computed`、`watch` 和生命周期钩子的基础用法。你可以把它当作 Vue Composition API 的最小复盘清单。

## Composition API

### `ref`

```ts
import { ref } from 'vue';

const count = ref(0);
count.value++;
```

### `reactive`

```ts
import { reactive } from 'vue';

const state = reactive({
  user: { name: 'Tom' },
});

state.user.name = 'Jerry';
```

### `computed`

```ts
import { computed, ref } from 'vue';

const firstName = ref('Ada');
const lastName = ref('Lovelace');
const fullName = computed(() => `${firstName.value} ${lastName.value}`);
```

### `watch`

```ts
import { ref, watch } from 'vue';

const keyword = ref('');

watch(keyword, async (val) => {
  // 处理请求、防抖等副作用
  console.log('search:', val);
});
```

## 生命周期（常用）

- `onMounted`：DOM 挂载后执行。
- `onBeforeUnmount`：组件销毁前清理定时器、监听器。
- `onUpdated`：响应式更新后的钩子（谨慎使用）。
